# `js-slack-bot` 🤖

## Usage

Use the `/cat-stmt` slash command to open the modal. Select a bank, enter an account name, and upload a bank statement CSV. The bot will call the categorizer and post the resulting Money Wiz CSV back to the channel.

Supported banks: **ASB**, **ANZ**, **Kiwibank**.

## Local dev

### Prerequisite

- install [ngrok](https://ngrok.com/)

### Commands

- Slack is unable to access to local dev server, so run this command to forward the request `ngrok http http://localhost:3002`, then copy the URL generated by ngrok and paste it to Slack bot configs's Event Subscriptions section.
